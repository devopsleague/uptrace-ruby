## Installation

```bash
gem install uptrace
```

## Introduction

uptrace-ruby is the official Uptrace client for Ruby that sends your traces/spans and metrics to
[Uptrace.dev](https://uptrace.dev).

See [uptrace-ruby documentation](https://docs.uptrace.dev/ruby/) for details.
